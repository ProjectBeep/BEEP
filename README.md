## 프로젝트

### 소개

- 다양한 기프티콘을 한 곳에 모아서 관리해보세요
- 기프티콘을 단순히 저장하고 보여주는 것만이 아닌 본인 위치를 기반으로 기프티콘을 추천해드려요
- 보안이 걱정이 된다면! 지문인증을 활용해서 기프티콘의 바코드를 숨겨보세요
